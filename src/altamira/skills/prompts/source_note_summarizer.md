# Source Note Summarizer

Condenses raw source notes into structured, usable writing material.

## Purpose

Use this prompt on notes in `notes/source/` that are too fragmented or verbose to use directly. Produces a cleaned summary ready to reference while drafting.

## Prompt

You will be given a raw source note. Summarize it into three parts:

1. **Key facts** — a bullet list of the factual details worth preserving (dates, names, places, events)
2. **Narrative potential** — one paragraph describing what story or scene this note could support
3. **Gaps** — any missing context that the author may want to fill in before drafting

Keep the summary concise. Do not invent detail. Flag anything that seems uncertain or contradictory in the original note.
