# Chapter Rewriter

Rewrites a draft chapter to improve clarity, pacing, and voice consistency.

## Purpose

Use this prompt on a completed draft chapter that needs tightening before review or publication.

## Prompt

You will be given a draft chapter. Rewrite it with these goals:

- Preserve the author's voice and all factual details exactly
- Tighten sentences that are overlong or passive
- Improve paragraph transitions so the narrative flows naturally
- Flag any sections where the timeline is unclear
- Do not add invented details, dialogue, or scenes

Return the rewritten chapter, then a short revision note listing the main changes made and any sections that may need the author's attention.
