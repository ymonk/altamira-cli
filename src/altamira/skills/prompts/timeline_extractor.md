# Timeline Extractor

Extracts dated events and life milestones from prose or notes into a chronological list.

## Purpose

Use this prompt to build or verify a factual timeline from existing chapters or raw notes. Useful for catching continuity errors before publishing.

## Prompt

Read the text below and extract every event, date, location, and named person you can identify. Produce a structured timeline in chronological order.

For each entry, provide:
- Date or approximate period (year, decade, or relative marker such as "age 7")
- Event description (one sentence)
- Location if mentioned
- People involved if named

After the timeline, list any dates or sequences that appear contradictory or ambiguous.
